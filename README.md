# GRAPH COLORING APP 



## About the project

This game allows you to test your knowledge on chromatic graph coloring!
Start the game and try to color the vertices as fast as you can but don't forget to use the least number of colors possible.

Throughout this game you will be able to find 3 game modes:

### TO THE BITTER END
Color the vertices in the order you want to but pay attention, you can't undo your colorings!

### RANDOM ORDER
Color the vertices starting from 1. You still cannot revert your colorings.

### I CHANGED MY MIND
Color the vertices starting from 1 but here you can use the undo button to revert your colorings!



## Getting started

Install the right version of java on your computer (22.0.2)
https://www.oracle.com/java/technologies/javase/jdk22-archive-downloads.html

Install the right version of javaFX on your computer (17.0.13)
https://gluonhq.com/products/javafx/

Run the project game via these versions.


## How to play

You can either generate a random graph, choose your own edges and vertices or generating from text file.
Some example of text files are in the example folder. Follow this structure to create your own one
